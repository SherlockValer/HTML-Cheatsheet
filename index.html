Metadata and Head Elements

Head Section Structure

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Page Title</title>
    <meta name="description" content="Page description">
    <meta name="keywords" content="keywords, for, search">
    <meta name="author" content="Author Name">
    <link rel="stylesheet" href="styles.css">
    <script src="script.js" defer></script>
    <link rel="icon" href="favicon.ico" type="image/x-icon">
</head>

Common Meta Tags

Meta Tag Description Example
charset Character encoding <meta charset="UTF-8">
viewport Responsive design control <meta name="viewport" content="width=device-width, initial-scale=1.0">
description Page description (SEO) <meta name="description" content="Page description">
keywords Keywords for search (less important now) <meta name="keywords" content="html, css, web">
author Page author <meta name="author" content="John Doe">
robots Control search indexing <meta name="robots" content="index, follow">
Open Graph Social media sharing info <meta property="og:title" content="Page Title">
Twitter Card Twitter sharing info <meta name="twitter:card" content="summary">
http-equiv HTTP header equivalents <meta http-equiv="refresh" content="30">

Link Types

Link Type Description Example
stylesheet Link to CSS file <link rel="stylesheet" href="styles.css">
icon Favicon <link rel="icon" href="favicon.ico" type="image/x-icon">
preconnect Preemptively connect to domain <link rel="preconnect" href="https://fonts.googleapis.com">
preload Preload resource <link rel="preload" href="font.woff2" as="font">
canonical Preferred URL (SEO) <link rel="canonical" href="https://example.com/page">
alternate Alternate version (language, RSS) <link rel="alternate" hreflang="es" href="https://es.example.com">

Script Element

<script src="script.js" defer></script>

<!-- Inline script -->
<script>
    function greet() {
        alert('Hello, world!');
    }
</script>

Best Practice: Use the defer attribute for scripts that need the DOM, or async for scripts that can run independently.

Global Attributes

These attributes can be used on any HTML element.

Attribute Description Example
id Unique identifier <div id="unique-id">
class Class name(s) for CSS/JS <div class="container blue">
style Inline CSS styles <div style="color: red;">
title Additional information (tooltip) <abbr title="HyperText Markup Language">HTML</abbr>
lang Language code <p lang="fr">Bonjour</p>
hidden Hide element <div hidden>Not visible</div>
dir Text direction <p dir="rtl">Right to left</p>
tabindex Tab order <button tabindex="1">First</button>
contenteditable Make element editable <div contenteditable="true">Edit me</div>
draggable Make element draggable <div draggable="true">Drag me</div>
data-* Custom data attributes <div data-user-id="123">User</div>
spellcheck Enable/disable spell checking <p spellcheck="true">Check spelling</p>
translate Whether to translate content <p translate="no">Don't translate</p>

Accessibility

ARIA Roles and Attributes

ARIA (Accessible Rich Internet Applications) helps make web content more accessible.

Common ARIA Roles

Role Description Example
role="button" Identifies element as a button <div role="button">Click me</div>
role="navigation" Navigation section <div role="navigation">Menu</div>
role="main" Main content <div role="main">Content</div>
role="search" Search functionality <form role="search">...</form>
role="alert" Important, time-sensitive info <div role="alert">Error message</div>

Best Practice: Use semantic HTML elements when possible instead of ARIA roles.

HTML Basics

HTML (HyperText Markup Language) is the standard markup language for creating web pages and web applications.

HTML Element Syntax

<tagname>Content goes here...</tagname>

Example:

<p>This is a paragraph.</p>

Self-closing Elements

<tagname attribute="value" />

Example:

<img src="image.jpg" alt="Description" />

Comments

<!-- This is a comment -->

Document Structure

Basic HTML Document

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document Title</title>
</head>
<body>
    <!-- Content goes here -->
</body>
</html>

Note: The <!DOCTYPE html> declaration tells the browser that the document is HTML5.

Text Elements

Headings

Element Description Example
<h1> Main heading (most important) <h1>Main Title</h1>
<h2> Secondary heading <h2>Subtitle</h2>
<h3> to <h6> Lower-level headings <h3>Section Title</h3>

Best Practice: Use headings in sequential order. Don't skip heading levels (like going from h1 to h3).

Links

Basic Link

<a href="https://www.example.com">Link text</a>

Link Attributes

Attribute Description Example
href Hyperlink reference (URL) <a href="page.html">Link</a>
target Where to open the linked document <a href="page.html" target="_blank">Link</a>
rel Relationship between current and linked document <a href="page.html" rel="nofollow">Link</a>
download Download the link instead of navigating <a href="file.pdf" download>Download</a>
title Additional information (tooltip) <a href="page.html" title="Visit Page">Link</a>

Link Types

Type Example
Absolute URL <a href="https://www.example.com">Link</a>
Relative URL <a href="page.html">Link</a>
Email link <a href="mailto:email@example.com">Email</a>
Phone link <a href="tel:+1234567890">Call</a>
Anchor link (page bookmark) <a href="#section-id">Go to Section</a>

Images and Media

Images

<img src="image.jpg" alt="Description of image">

Best Practice: Always include the alt attribute for accessibility. It should describe the content of the image.

Image Attributes

Attribute Description Example
src Source URL of the image <img src="image.jpg">
alt Alternative text description <img src="image.jpg" alt="Description">
width Width of the image (in pixels) <img src="image.jpg" width="300">
height Height of the image (in pixels) <img src="image.jpg" height="200">
loading Lazy loading for images <img src="image.jpg" loading="lazy">

Image Maps

<img src="image.jpg" alt="Map" usemap="#mapname">
<map name="mapname">
    <area shape="rect" coords="x1,y1,x2,y2" href="url1" alt="Link 1">
    <area shape="circle" coords="x,y,radius" href="url2" alt="Link 2">
</map>

Picture Element

<picture>
    <source media="(min-width: 650px)" srcset="large.jpg">
    <source media="(min-width: 450px)" srcset="medium.jpg">
    <img src="small.jpg" alt="Description">
</picture>

Audio

<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
    <source src="audio.ogg" type="audio/ogg">
    Your browser does not support the audio element.
</audio>

Video

<video width="320" height="240" controls>
    <source src="video.mp4" type="video/mp4">
    <source src="video.webm" type="video/webm">
    Your browser does not support the video element.
</video>

Embedding Content

<iframe src="https://www.example.com" width="500" height="300"></iframe>

Tables

Basic Table Structure

<table>
    <caption>Table Caption</caption>
    <thead>
        <tr>
            <th>Header 1</th>
            <th>Header 2</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Row 1, Cell 1</td>
            <td>Row 1, Cell 2</td>
        </tr>
        <tr>
            <td>Row 2, Cell 1</td>
            <td>Row 2, Cell 2</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <td>Footer 1</td>
            <td>Footer 2</td>
        </tr>
    </tfoot>
</table>

Table Cell Attributes

Attribute Description Example
colspan Number of columns a cell should span <td colspan="2">Content</td>
rowspan Number of rows a cell should span <td rowspan="3">Content</td>
headers Associates data cells with header cells <td headers="header-id">Content</td>

Table Element Roles

Element Description
<caption> Table caption/title
<thead> Groups header content
<tbody> Groups body content
<tfoot> Groups footer content
<tr> Table row
<th> Table header cell
<td> Table data cell
<colgroup> Groups columns for formatting
<col> Specifies properties for columns

Forms

Basic Form Structure

<form action="/submit" method="post">
    <fieldset>
        <legend>Form Section</legend>
        <label for="name">Name:</label>
        <input type="text" id="name" name="name" required>
        
        <input type="submit" value="Submit">
    </fieldset>
</form>

Form Attributes

Attribute Description Example
action URL where form data is sent <form action="/submit">
method HTTP method (get or post) <form method="post">
enctype How form data is encoded <form enctype="multipart/form-data">
autocomplete Enable/disable autocomplete <form autocomplete="off">
novalidate Disable browser validation <form novalidate>

Element Description Example
